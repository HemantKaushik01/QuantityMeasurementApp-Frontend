# Quantity Measurement App Frontend

## Project Overview
This project is the frontend for the Quantity Measurement App, designed to assist users in converting and measuring different units of quantity.

## Technology Stack
- **JavaScript**: 59%  
- **CSS**: 39.2%  
- **HTML**: 1.8%  

## Features
- User-friendly interface for quantity conversion.
- Real-time conversion calculations.
- Responsive design for both mobile and desktop users.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/HemantKaushik01/QuantityMeasurementApp-Frontend.git
   ```
2. Navigate to the project directory:
   ```bash
   cd QuantityMeasurementApp-Frontend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Backend Reference
For the backend implementation, refer to the repository: [Quantity Measurement App Backend](https://github.com/HemantKaushik01/QuantityMeasurementApp)

## Deployment Link
You can view the deployed application at: [Quantity Measurement App Frontend](https://quantity-measurement-app-frontend-peach.vercel.app/)

## Usage Guide
1. Open the deployed application link.
2. Select the quantity type you want to convert.
3. Enter the value and select the source unit.
4. Select the target unit.
5. The converted value will be displayed instantly.
